Vue3 安装与配置完全指南

Vue.js 是目前最流行的前端框架之一。本教程将带你从零开始完成 Vue3 的安装与配置。

--- ## 📋 前置条件 在开始安装 Vue3 之前，请确保你的开发环境满足以下要求：

Node.js >= 16.0 已安装

npm >= 7.0 或 pnpm 已安装

已安装 VS Code 或其他代码编辑器

已安装 Git

推荐使用 nvm（Node Version Manager）来管理 Node.js 版本，方便在不同项目之间切换。
